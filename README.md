# pt
Prints quick, useful information about elements on the periodic table, all in your shell.

## Installation
```sh
sudo make install
```

## Usage
```sh
pt -[hns] [input]
```

`-h`, `--help`: Print this help message\
`-n [Atomic Number]`: Get info for [Atomic Number]\
`-s [Symbol]`: Get info for element [Symbol]
